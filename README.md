## 🚀 Getting Started

### Prerequisites

- Python 3.9+

### Installation

1.  Clone the repository and switch to the `python` branch:
    ```bash
    git clone <repo_url>
    cd BitcoinWallet
    git checkout python
    ```

2.  Create and activate a virtual environment:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
